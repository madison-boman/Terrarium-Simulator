# Sprite assets

Place your game art here with these exact names:

- `snail-sprite.png` (4x4 sprite sheet rows: moving, turning, idle, dead)
- `jar-sprite.png` (2-frame sprite sheet: sealed jar, open jar)
- `water-effects.png` (water/humidity effect sheet)

The React app loads these at runtime and will automatically switch from fallback
graphics to your provided sprites.
